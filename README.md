# 🏖️ Smart Leave Management System

A ServiceNow-based application that automates employee leave requests, approval workflows, leave day calculations, and request tracking.

## ✨ Features

- Leave Request Submission
- Automated Manager Approval
- Auto Leave Days Calculation
- Date Validation using Client Script
- Approval Status Tracking
- Centralized Leave Management

## 🛠️ Technologies Used

- ServiceNow App Engine Studio
- Flow Designer
- Business Rules
- Client Scripts
- Custom Tables

## 📋 Workflow

Employee Creates Leave Request  
↓  
Pending Approval  
↓  
Manager Approval  
↓  
Approved / Rejected  
↓  
Status Updated Automatically
